# Tools for optical lab

This git create a PWA (Progressive Web app) `CageCam`.
The app can be locally installed on the phone and used offline.

## CageCam WebApp
This tool control miniature esp32 camera module and display the image on the webpage.
Wifi connection is used for image transfer. The camera create own wifi.

The camera should allow better visual check during aliment of optical components.
It is for use in optical laboratory.

Link : https://ondrejstranik.github.io/CageCam/
